# 现实版躲猫猫游戏工具 - Peekaboo Game

一个基于地理位置和P2P技术的多人现实躲猫猫游戏工具。玩家分为"猫"和"老鼠"两个角色，在现实世界中展开捉迷藏游戏。

## 游戏特色

- 🌍 **基于地理位置**：使用GPS蓝牙wifi等精准定位，在现实世界中游戏
- 🔗 **P2P网络架构**：使用WebRTC技术，无需中央服务器
- 🗣️ **实时语音聊天**：支持多人语音通话
- 📍 **高德地图集成**：实时显示玩家位置
- 📱 **响应式设计**：支持手机和桌面设备
- ⚡ **实时性能监控**：显示网络延迟和定位精度

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **地图服务**: 高德地图API
- **P2P通信**: 使用PeerJS构建网形网络
- **部署**: GitHub Pages (纯静态网站)




## 游戏玩法

### 角色说明

- **🐱 猫 (Cat)**: 追捕者，可以看到地图和所有玩家位置
- **🐭 老鼠 (Mouse)**: 躲藏者，没有地图显示，只能看到自己的状态



1. 组建网络，组建p2p网形网络，每个玩家都是一个节点，节点之间通过P2P技术通信。玩家昵称与Peer ID
2. **角色分配**: 投票选出猫，可以弃权
3. **准备**: 所有玩家准备好后，游戏开始
到game.html
4. **躲藏阶段**: 老鼠寻找躲藏地点，猫等待倒计时
5. **寻找阶段**: 猫开始寻找老鼠，可以定时扫描老鼠位置，被抓住老鼠的转换为猫，并实时更新
6. **游戏结束**: 所有老鼠被抓住时游戏结束

### 核心功能

- **扫描功能**: 只有猫在扫描可用期间扫描并在地图上显示老鼠位置
- **抓住机制**: 猫可以发送抓住请求，老鼠确认后转换为猫
- **位置追踪**: 实时显示玩家位置（猫可见）
- **语音聊天**: 支持网内语音通话
- **性能监控**: 实时显示网络延迟和定位精度
